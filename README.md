# cattle_mutilation
M5stack用にコンパイルしてご使用ください。

主人公の牛を操作して草を食べるゲーム。草をたくさん食べてハイスコアを目指そう。<br>
空に浮かぶUFOからはキャプチャービームがだ出され、ビームに触れるとさらわれるので注意してね。<br>
３回さらわれるとゲームオーバーです。<br>
Aボタン(左ボタン)：草を食べる　　Cボタン（右ボタン）：歩く<br>
